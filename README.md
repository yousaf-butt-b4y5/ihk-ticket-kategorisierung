# IHK-Abschlussprojekt FISI (AP2)

**Thema:** Konzeption und Implementierung einer automatisierten Ticket-Kategorisierung und Priorisierung im IT-ServiceDesk unter Einsatz von PowerShell

**Prüfling:** Yousaf Butt  
**IHK:** Nürnberg für Mittelfranken  
**Prüfungstermin:** Sommer 2026  
**Projektumfang:** 40 Stunden

## Projektziel

Automatisierte Verarbeitung von Ticketdaten mittels PowerShell:
- Einlesen von Ticketdaten aus CSV
- Keyword-basierte Kategorisierung
- Berechnung von Urgency und Impact
- Priorisierung (P1–P5) per Matrix
- Strukturierte Ausgabe als CSV
